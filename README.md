💱 Currency Converter Chatbot

A smart chatbot that allows users to convert currencies in real time while also supporting casual conversations for an interactive and engaging experience. The chatbot is built using Dialogflow for natural language understanding and Flask API for backend integration.

🚀 Features
🌍 Convert one currency to another using simple text commands.
💬 Engage in casual conversations for a more interactive user experience.
🤖 Natural language processing powered by Dialogflow.
🔄 Real-time currency conversion through API integration.
⚡ Fast and lightweight Flask backend.
📱 Easy to deploy and integrate with web applications.
🛠️ Technologies Used
Dialogflow – Intent recognition and conversational AI.
Flask – Backend REST API.
Python – Core programming language.
Currency Exchange API – Real-time exchange rate retrieval.
JSON – Data exchange between Dialogflow and Flask.
📂 Project Workflow
User sends a message to the chatbot.
Dialogflow detects the user's intent.
If the request is for currency conversion, Dialogflow forwards it to the Flask webhook.
Flask fetches the latest exchange rates from a currency API.
The converted value is returned to the chatbot.
For general conversations, Dialogflow responds directly based on predefined intents.
